🐱 Spinning Cat Loader

This is a pure-CSS animated cat that spins while your page loads. It was extracted from hexo-theme-shoka. No images required — the entire cat is drawn with CSS gradients, clip-paths, and borders.

Manual Controls

Press these buttons to test the loader API:

You can also click on the cat to dismiss it. Hover over it to pause the animation.

Features

  • 🐱 Pure CSS orange tabby cat — zero images
  • 🔄 Spins 720° while the body "morphs" via clip-path
  • 🖱️ Hover to pause, click to dismiss
  • 👁️ Re-shows when user switches to another tab (optional)
  • 📦 ~300 lines of CSS, ~100 lines of JS
  • 🎬 Optional smooth fade with anime.js

How to Integrate

  1. Copy <div id="loading">...</div> right after <body>
  2. Include spinning-cat-loader.css
  3. Include spinning-cat-loader.js (and optionally anime.js)
  4. Configure via CatLoaderConfig (see README)
